# 🚀 发布 EmotiChat：现代化的情感陪护聊天客户端

大家好！我开发了 **EmotiChat**，一个基于 **Next.js 15 + React 19** 构建的现代化 AI 聊天客户端。

**它是什么？**
一个类 SillyTavern 的项目，专注于**情感陪护**与**角色扮演**，但采用了完全现代化的技术栈和工程架构。

**与 SillyTavern 的核心差异：**
*   **轻量化与现代化**：抛弃了旧时代的包袱，代码高度解耦，运行更流畅。
*   **无兼容层**：**我们不提供 SillyTavern 的数据兼容或转换**。这是一个全新的开始，专注于更好的架构设计。

**特色功能：**
*   🧠 **硬核提示词工程**：支持楼层（Layer）与深度（Depth）概念，可精确控制提示词注入位置（Before/After/Replace）。
*   🎨 **现代化 UI**：基于 shadcn/ui，支持 Markdown、KaTeX 公式、代码高亮及 HTML 预览。
*   🔧 **开发者友好**：内置强大的 Dev Mode，可实时查看构建后的 Prompt、Token 统计及原始 API 日志。
*   🔒 **隐私优先**：所有数据（API Key、聊天记录）仅存储在本地文件系统或浏览器中。
